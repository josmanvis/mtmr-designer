# MTMR 2026

> **A customized version of [MTMR](https://github.com/Toxblh/MTMR) bundled with MTMR Designer**

🌟 My TouchBar My rules - The Touch Bar Customisation App for your MacBook Pro

## About This Fork

This is a customized version of MTMR bundled with the [MTMR Designer](https://github.com/mtmr-designer) project. 

**Version:** 2026.1  
**Bundle ID:** `com.mtmr-designer.mtmr2026`

## Changes from Upstream

- Rebranded as "MTMR 2026"
- Updated bundle identifier for MTMR Designer distribution
- Updated version numbering scheme (2026.x)

## Building

1. Open `MTMR.xcodeproj` in Xcode
2. Select the "MTMR" target
3. Build (⌘B) or Archive for distribution

### Requirements

- macOS 10.12.2 or later
- Xcode 10.0 or later
- Swift 5.0

## Original MTMR Documentation

For full documentation, see the [original MTMR repository](https://github.com/Toxblh/MTMR).

### Basic Usage

1. Build and run MTMR 2026
2. The app will create a default preset at `~/Library/Application Support/MTMR/items.json`
3. Customize your Touch Bar by editing this JSON file
4. Use [MTMR Designer](https://github.com/mtmr-designer) for a visual editor
